## Development

### Deployment
確定你有 php tidy, php curl

1. Clone the Repo
2. Import `.sql`
3. `touch magic.php` and set the following
```PHP
<?php
	const USERNAME = "your_sql_username";
	const PASSWORD = "your_sql_password";
	const USOCKET = "your_sql_socket";
?>
```
