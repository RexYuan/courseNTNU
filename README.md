## Development

### Deployment

1. Clone the Repo
2. Import `.sql`
3. `touch magic.php`: remember to set USERNAME and PASSWORD, e.g.
```
<?php
	const USERNAME = "your_username";
	const PASSWORD = "your_password";
?>
```

### Style
- use Scss
```
scss --watch ./
```
